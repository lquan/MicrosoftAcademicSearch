Now consider again Microsoft Academic Search. When the (autocompletion suggested) query ``traveling tournament problem'' is entered, the eye-catcher is a graph with the number of publications in function of the time. The graph shown is rather big and less relevant than the search results for a normal user who wants to search information.

Figure 5: Search results of Microsoft Academic Search for the query ``traveling tournament problem''.

This query only returns 5 results, all published in 2010, but `feel' less important overall (see ttpproblem). The results page gives an explanation of this behavior, but it is rather vague (``This page shows one keyword best matching your query, you can find other results here''). After clicking on that link, the results shown are again the same as in ttp. This is rather unintuitive behavior and could be better explained or handled.
